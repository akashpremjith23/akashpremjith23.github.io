# Aveeva Herold CV Website

A simple, editorial-style CV website built from Aveeva Herold's 2026 CV. This site includes:

- Responsive layout with glassmorphism and gradient styling
- Smooth scrolling navigation
- Mobile-friendly menu and intuitive sections
- Future-ready visual design for 2027 and beyond

## Open in Visual Studio or Visual Studio Code

1. Open this folder in Visual Studio or Visual Studio Code.
2. Open `index.html`.
3. If you have the Live Server extension installed in Visual Studio Code, right-click the page and choose `Open with Live Server`.
4. If you do not have Live Server, open `index.html` directly in your browser.

## Optional Local Server

If you want a local development server, you can run one from the terminal in this folder:

- With Python 3:
  ```bash
  python -m http.server 5500
  ```

- Then open: `http://localhost:5500`

## Publish without hosting it yourself

If you want the website to be available on a phone or any browser without running a server yourself, GitHub Pages is a great free option:

1. Create a new repository on GitHub.
2. Push this folder's files to the repository's `main` branch.
3. In the repository settings, enable GitHub Pages from the `main` branch and the root folder.
4. GitHub Pages will provide a public URL like `https://<username>.github.io/<repo-name>`.

### Quick phone access without publishing

If you just want the site on a phone without a hosted URL:

- Zip the folder and send it to the phone.
- Open `index.html` in the phone browser.

## Files

- `index.html` — main website structure
